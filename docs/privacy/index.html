privacy policy

Do's

debug logging

some debugging data will be logged, for a maximum of 7 days. this data will never include any ip addresses, or other unneeded identifying data, and will never be collected or analyzed for tracking purposes.

public data

data that gets saved online and publicly in our database:

  1. signing git username
  2. signing public gpg key
  3. git provider (gitlab, github, ...)
  4. domain
  5. git org
  6. repository
  7. a list of files this homepage is expected to serve and their hashes.
  8. a comment by the publisher, informing about the changes.

this is all the data we save, there is no hidden database, everything we collect is also public by default because it is needed.

all authentication is done locally using the local gpg installation, no login to our servers is possible or necessary. we will run tor proxies to allow anonymous publishing.

Dont's

we will never, with or without consent,

  1. track user activity on our servers.
  2. sell information about our users.
  3. pass on any information about our users on to any third parties.

canary:

if this notice disappears, it means that one of the rules above has been broken.

local data

personal data we access on your machine and do not send anywhere, ever:

  • your git config user.email (never uploaded)
  • your public ssh key associated with that email.
  • your gpg keys published on github or gitlab.

thats it.
